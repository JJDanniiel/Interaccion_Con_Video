Proyecto Final

Este proyecto propone una experiencia inmersiva donde los jugadores interactúan con un entorno virtual a través de movimientos de las manos y sonidos del entorno. Inspirado en la convergencia entre cuerpo y tecnología, el juego fomenta una conexión multisensorial, permitiendo a los jugadores utilizar gestos y comandos sonoros como herramientas principales para superar desafíos.

El diseño del juego se centra en la interacción intuitiva, combinando precisión técnica con creatividad para ofrecer mecánicas novedosas y accesibles. A medida que los jugadores avanzan, los gestos se vuelven más complejos, y los sonidos que emiten (como aplausos, chasquidos o palabras clave) desbloquean nuevas habilidades, alteran el entorno o activan eventos únicos.

Detección de manos con ml5.js:

Se emplea la tecnología de Handpose para rastrear en tiempo real las posiciones clave de las manos, permitiendo realizar acciones como mover elementos, seleccionar objetos y crear trayectorias dinámicas.

Clasificación de sonidos con ml5.js:

Se implementa el modelo de clasificación de sonidos (SoundClassifier) para interpretar entradas sonoras del usuario, transformándolas en comandos específicos que complementan la interacción gestual.
